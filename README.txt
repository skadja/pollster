== Description ==

Pollster is a FREE WordPress theme for measuring likeability or popularity between two contestants. It's easy to use and fully customizable via the WordPress Theme customization section.

For more information about Pollster please go to https://github.com/skadja/pollster.

== Installation ==

1. Download Pollster.
2. Unzip the compressed file on your computer.
3. Log in to your site's WordPress Admin.
4. Go to Appearance > themes. Click on the Add New at the top of the screen, then click on Upload Theme.
5. Click on the Browse button to select the zip file from your computer, then click Install Now.
6. Navigate to Appearance > Customize in your admin panel and customize to taste.
